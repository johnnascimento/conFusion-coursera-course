Introduction:

The main goal of this website is to provide people with localization features, like google maps. However, different from google, it'll not only show the places around and mark the path where from to go but will also show the users suggestion of great places surrounding them, based on their current location.

Expected list of features:

  • Localization with gps
  • accelerometer sensor for identifying towards with way * * the user is directing
  • Notifications based on users intrests
